### 界面重构规划

**现状问题：** 双栏布局（左 2/3 + 右 1/3），处理进度常驻右侧空占位，AI 配置和上传文件占据左栏黄金位，字幕表格 + AI 推荐 + 开始处理全挤在同一卡片。

**目标：** 改成单列三步纵向布局，视线和操作直线向下——准备 → 筛选 → 生成。

```
┌──────────────────────────────────────────┐
│  Navbar (不改)                            │
├──────────────────────────────────────────┤
│  使用说明 (可折叠，不改)                   │
├──────────────────────────────────────────┤
│  Step 1 · 准备素材                        │
│  ┌──────────────┬──────────────────────┐ │
│  │ 📹 视频上传   │  AI 配置 (一行摘要)   │ │
│  │ [生成字幕]   │  api.deepseek.com    │ │
│  │ 📄 字幕上传   │  ***sk-xxxx [展开]   │ │
│  └──────────────┴──────────────────────┘ │
├──────────────────────────────────────────┤
│  Step 2 · 筛选内容                        │
│  ┌──────────────────────────────────────┐│
│  │ [AI 推荐] [仅选推荐] [提示词 ▾]      ││
│  │ ┌──────────────────────────────────┐ ││
│  │ │ 字幕表格 (全宽)                  │ ││
│  │ └──────────────────────────────────┘ ││
│  │              [开始处理 (15 条)]      ││
│  └──────────────────────────────────────┘│
├──────────────────────────────────────────┤
│  Step 3 · 生成卡片 (仅处理时显示)         │
│  ┌──────────────────────────────────────┐│
│  │ 进度条 ████████░░ + 步骤提示         ││
│  │ 卡片预览 (左右翻页)                  ││
│  │            [下载 .apkg]              ││
│  └──────────────────────────────────────┘│
└──────────────────────────────────────────┘
```

**具体改动：**

| 原来 | 改后 |
|------|------|
| 双栏（左 2/3 右 1/3） | **单列三步纵向** |
| AI 配置占整个卡片 | 折叠成一行摘要，放 Step 1 右上角 |
| 上传文件独立卡片 | 和 AI 配置合并到 Step 1，横向排列 |
| 处理进度常驻右侧 | 归入 Step 3，仅处理时显示 |
| 字幕表格挤在左栏窄空间 | **全宽展开**，Step 2 独占 |
| 按钮到处散落 | 每步底部一个主按钮 |

**不改的部分：** 顶部导航栏（标题 + 使用说明 + 反馈）、使用说明面板（基础/进阶双版本）保持现有设计。
