### 三、ffmpeg 路径处理

打包后 exe 的同级目录结构和开发时不同，需要动态查找：

```python
# utils.py
import sys, os

def get_bin_path(name: str) -> str:
    if getattr(sys, "frozen", False):
        # 打包后：exe 所在目录
        base = os.path.dirname(sys.executable)
    else:
        # 开发时：项目根目录
        base = os.path.dirname(os.path.abspath(__file__))
    return os.path.join(base, "bin", name)

FFMPEG  = get_bin_path("ffmpeg.exe")
FFPROBE = get_bin_path("ffprobe.exe")
```

所有调用 ffmpeg 的地方换成 `FFMPEG` 和 `FFPROBE` 变量，不要硬编码字符串。

### 四、PyInstaller .spec 文件

```python
# anki_maker.spec
a = Analysis(
    ["main.py"],
    pathex=[],
    binaries=[
        ("bin/ffmpeg.exe",  "bin"),
        ("bin/ffprobe.exe", "bin"),
    ],
    datas=[],
    hiddenimports=[
        "genanki",
        "pysrt",
        "openai",
    ],
    hookspath=[],
    noarchive=False,
)

pyz = PYZ(a.pure, a.zipped_data)

exe = EXE(
    pyz,
    a.scripts,
    [],
    exclude_binaries=True,
    name="AnkiMaker",
    icon="assets/icon.ico",
    console=False,           # 有 GUI 关掉控制台；纯命令行改为 True
)

coll = COLLECT(
    exe,
    a.binaries,
    a.zipfiles,
    a.datas,
    name="AnkiMaker",
)
```

执行打包：

```bash
pyinstaller anki_maker.spec
```

输出在 `dist/AnkiMaker/`，是一个文件夹。

### 五、Inno Setup 脚本

```ini
; installer.iss
[Setup]
AppName=Anki 卡片生成器
AppVersion=1.0
AppPublisher=你的名字
DefaultDirName={autopf}\AnkiMaker
DefaultGroupName=AnkiMaker
OutputBaseFilename=AnkiMaker_Setup
OutputDir=installer
Compression=lzma2/ultra64
SolidCompression=yes
; 要求管理员权限（写入 Program Files 需要）
PrivilegesRequired=admin

[Files]
Source: "dist\AnkiMaker\*"; DestDir: "{app}"; Flags: recursesubdirs

[Icons]
Name: "{group}\Anki 卡片生成器"; Filename: "{app}\AnkiMaker.exe"
Name: "{userdesktop}\Anki 卡片生成器"; Filename: "{app}\AnkiMaker.exe"

[Run]
Filename: "{app}\AnkiMaker.exe"; Description: "立即启动"; Flags: postinstall skipifsilent

[UninstallDelete]
; 卸载时清理用户生成的 output 目录
Type: filesandordirs; Name: "{app}\output"
```

用 Inno Setup 编译后得到 `installer/AnkiMaker_Setup.exe`。

### 六、完整执行流程

```
1. pip install pyinstaller openai genanki pysrt
2. 下载 ffmpeg 静态编译版，解压 ffmpeg.exe / ffprobe.exe 到 bin/
3. pyinstaller anki_maker.spec
4. 用 Inno Setup 编译 installer.iss
5. 在一台干净的 Windows 机器上测试 AnkiMaker_Setup.exe
```

ffmpeg 静态编译版下载地址：https://github.com/BtbN/FFmpeg-Builds/releases，选 `ffmpeg-master-latest-win64-gpl.zip`。
